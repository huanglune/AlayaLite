# Full-cache adjudication probes (historical)

This directory is intentionally retained in place as the executable evidence
bundle for the full-cache and topology-preserving-seal decisions. It is not a
current benchmark suite.

Several arms, especially `bench_memqg.py`, reference Python index APIs and
memory-QG serving paths that have since been retired. Do not use them as
examples of the supported `flat`/`qg` Collection surface. Reproduction requires
the recorded historical revision and the original external datasets/hosts;
current maintenance should use the active targets under `benchmarks/`.
