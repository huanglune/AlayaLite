# Full-cache adjudication probes (historical)

This directory is intentionally retained in place as the executable evidence
bundle for the full-cache and topology-preserving-seal decisions. It is not a
current benchmark suite.

The numerical comparison remains historical, but `bench_memqg.py` now uses the
current `connect`/QG Collection surface so repository-wide SDK examples do not
preserve a second API generation. Reproduction still requires the recorded
external datasets and hosts; current maintenance should use the active targets
under `benchmarks/`.
